Quick Setup
-----------
``` shell
git clone https://github.com/cadejscroggins/cadejscroggins.github.io
cd cadejscroggins.github.io
git checkout dev
bower install && npm install
grunt serve # for editing
grunt deploy # to deploy to the master branch
```
